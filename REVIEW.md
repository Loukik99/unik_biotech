## Installation Steps
The application lives in the `frontend/` directory (Create React App + CRACO).

1. `cd frontend`
2. `npm install`
3. `npm start`

> Note: this project uses Create React App (via CRACO), so the local dev server
> command is `npm start` (there is no `npm run dev`). To generate a production
> build, run `npm run build`.
